# unsafe_allow_html=True 使用箇所一覧

第2段階の安全対策として、`app.py` 内の `unsafe_allow_html=True` 使用箇所を一覧化しました。

## 方針

- CSS注入や固定レイアウト用HTMLは画面構成維持のため残します。
- ユーザー入力や記録本文がHTML内へ入る箇所は `html_escape_text()` でエスケープします。
- AI、PDF、写真、バックアップ、Supabase同期には触れません。
